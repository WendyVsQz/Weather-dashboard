# Weather dashboard

This weather dashboard allows users to search for a city to get the current weather and 5 day forecast. It will display the the city name, the date, the temperature, the humidity and the wind speed. 

## Features
* Bootstrap
* JQuery
* Moment.js
* Google fonts
* Font awesome
